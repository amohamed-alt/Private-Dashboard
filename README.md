# Retention Command Center UI V2

This package updates the dashboard UI only.

## What changed

- No calculation changes.
- No n8n changes.
- No `data/live-data.js` changes.
- Retention-only view.
- Acquisition is hidden from the interface.
- Modern dashboard layout inspired by executive CRM dashboards.
- New sections:
  - Overview
  - Forecast View
  - Fadi
  - Jihad
  - Faizan
  - Plan vs Calculated
  - Accounts Explorer
  - Action Center
  - Data Quality

## Upload to GitHub

Replace these files in the repo root:

```text
index.html
style.css
app.js
```

Keep this file generated by n8n:

```text
data/live-data.js
```

## Notes

The UI reads existing generated values from `window.DASHBOARD_DATA`. It does not change formulas.
