# Retention Command Center

Static GitHub Pages dashboard for Retention only.

## Scope

This dashboard intentionally ignores Acquisition display and focuses only on:

- Fadi
- Jihad
- Faizan

Data source:

```text
data/live-data.js
```

This file is generated by n8n from Google Sheets every 12 hours.

## Required files

```text
index.html
style.css
app.js
data/live-data.js
```

## What the dashboard shows

- Retention Overview
- Fadi page
- Jihad page
- Faizan page
- Plan vs Calculated
- Accounts
- Action Center
- Data Quality

## Forecast logic

```text
Worst = Confirmed Revenue + Pending High
Medium = Worst + Pending Medium
Best = Medium + Pending Low
Best + Lost = Audit view only
```

Lost is not counted as positive revenue in the default Best forecast.
